Conclusiones modelos Matematico y probabilistico¶

Mejor prediccion

  • A mi criterio el motelo amtematico en concreto el modelo polinomial, vemos que tienes ms coincidencias con respecto a los datos por lo que con este modelo podemos obtener un resultado bueno, asi mismo al prediccion que hace va ajustarse a lo que podria apsar.

Ventajas

  • modelo matematico
      • Es una manera de ver como se compartan los datos.
      • Se puede tener de manera grafica como afecta un evento.
      • Se tiene diferentes modelos apra cada tiempo demuestra

Desventajas

  • modelo matematico
      • En ciertas ocaciones algunos modelos fallan
      • la mayoria de ellos no son completamente exactos

Comparacion

El modelo matematico busca determinar la relacin entre una variable dependiente Y con respecto a otras variables llamadas explicativas o independientes X. Asimismo, el modelo busca determinar cul ser el impacto sobre la variable Y ante un cambio en las variables explicativas X.

EL modelo probabilistico es un modelo matematico el cual estudia resultados de un supuesto caso aleatorio para ver como se va a ir comprtanto en el futuro y asi obtener un supuesto resultado para dar una solucion.
